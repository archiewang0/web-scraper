# 網頁爬蟲使用手冊

## 簡介

本手冊提供網頁爬蟲的基本知識和實踐指南

## 系統功能

- **資料獲取**：從使用者自己設定的目標網頁
- **爬蟲內容設定**：選擇網頁內容的爬蟲方式（如文字、圖片）
- **視覺化展示**：提供卡片形式來呈現爬蟲資訊

## 開始使用

### 首頁 (landing page)

首頁提供系統的基本介紹和主要功能入口。你可以從這裡直接進入設定頁。

### 設定頁面 (Setting page)

設定頁面列出了系統中所需要的設定項目。你可以根據需要進行修改。

1. **網頁URL爬蟲目標**：輸入網頁URL作為爬蟲目標。
2. **🐛🐛名稱**：可以作為爬蟲結果的標題。
3. **Content Selectors (CSS selectors)**：輸入網頁元素選擇器，用於獲取爬蟲結果。

### 爬蟲結果頁 (Result page)

爬蟲結果頁顯示了系統從目標網站中提取出來的內容。
